# Key mappings
Aqui anotarei os atalhos mais importantes da minha configuração

## Indice
<M> = alt
<C> = ctrl
<leader> = space

## Navegação
navegar para o buffer/linha anterior = <C-o>
navegar para o buffer/linhagit push posterior = <C-i>

## Mapeamentos importantes
Undo = U
Redo = <C-R>

## LSP
Ver definição do método = K
Abrir linha de diagnostico = grl

## Busca
ver opções de busca = <leader>s

## Autocomplete
Aceitar <C-y>

## Tabs (bufferline)
Ir para o buffer 1 g1
Ir para o buffer 2 g2
Ir para o buffer n gn
Ir para o último buffer g0
Ir para o próximo buffer <M-j>
Ir para o buffer anterior <M-k>
Mover para o buffer para trás <M-J>
Mover para o buffer para frente <M-K>
